# Meiktila Local Service and Information System (Meiktila Hub)

A web-based digital platform designed to connect residents and visitors in Meiktila with essential local service providers, improving data accessibility and community convenience.

## 🚀 Features
- Admin Dashboard: Centralized panel for managing system statistics and configurations.
- Provider Management (CRUD): Fully functional system to Create, Read, Update, and Delete service provider profiles.
- Service Category Management (CRUD): Structured categorization of local services (e.g., repairs, transport, health) for seamless navigation.
- Automated Database Logic: Utilizes optimized MySQL database triggers to ensure data integrity and seamless backend performance.

## 🛠️ Tech Stack
- Backend: PHP / Laravel Framework
- Database: MySQL
- Frontend: HTML, CSS, JavaScript, Bootstrap

## 📂 Database Architecture
The system includes structured tables for providers, services, and categories, interconnected with relational constraints and triggers to manage real-time updates efficiently.
